What is a paradigm?

There are many ways to write a text with a language. In short a language has only words and rules on how to use them . The use of these words and rules is at the mercy of the writer. So, for example, with Portuguese you can write scientific as well as chronic texts. The scientific will be a descriptive text and poetry will be an extremely subjective text. The two texts will be totally different but they will be using the same language. So a paradigm can be defined with a way of writing the code. And to write such a code will be given a set of tools.

What is the object-oriented paradigm in Python?

A paradigm in computer science is usually a method of categorizing the resources that a language offers you. So, for example, Python is a language that has the paradigm of object orientation. For it to be classified as such it must offer you 3 resources: inheritance, polymorphism, encapsulation. These three features are used to ORGANIZE the codes in classes.

When you program using the object orientation paradigm in Python all your effort is in using the three available resources to organize your code. Programming using the object orientation paradigm is different from programming using the procedural paradigm. In the procedural paradigm you are using the capabilities of the paradigm to concatenate the different procedures. Generally breaking the general goal (e.g. going from point A to point B) in small logical steps. In the object orientation paradigm, your effort is to organize all actions and data into classes.
