Atom as IDE for Python code.

18 de April de 201811 de November de 2018 JoaoMarcolanpractice

Programs created in Python are highly dependent on the indentation found in the text of the document. A text editor that helps you assemble the required structure can be useful at the time of programming. Come and check out this tutorial where I will introduce how to make Atom IDE for Python. Better how to make Atom as the best IDE for Python. Including making him autocomplete Python code.  This is the IDE used in object orientation tutorials for Python . So, how is the IDE used in the introductory tutorials on programming in Python .

Atom

Using a proper IDE can increase the performance of programmers in any language. Using a text editor that finds some tools (e.g., auto-complete and document-driven) improves programmers’ productivity. Atom is a modern text editor that can be completely customized and can be used to program in many different programming languages.

Figura 1. Possible atom settings: kernel, text editor, shortcut keys, themes, packages.

 

If it is your first time using this editor, it will display the welcome file on the home screen and with this document you can reach the settings document (figure 1).

Figure 2. Atom Welcome File

If you closed the welcome tab you could always reopen the settings document by going to file> settings .

Modifying Atom to become the best IDE for Python

Let’s set up Atom to be an IDE for Python. Atom has thousands of packages that can add new features. In this tutorial we will install some packages that interest us and the keyboard shortcuts can also be customized. In the atom the shortcut keys are called keybinds . In this tutorial we will present some shortcuts of the atom that are already configured and some that comes in the installation of the new plugins. Atom also has the functionality of modifying the layout of the text editor. I particularly like the black layout but you can set it up any way you want. OK! let’s make Atom an IDE for Python.

Adjusting the text editor

The first thing we will adjust is the text editor to be optimized for Python. To do this, go to the configuration file (figure 1) and click the editor tab. The first setting is to check the boxes: number and show ident Guide These two settings serve to make it easier to visualize the indentations in the files. set the value 4 in the Tab length field.This setting means that each time you press the tab key, the text editor places 4 spaces, the value 4 has been defined in the PEP 8. You can view these two configurations in figure 3.

Figure 3. Atom text editor configuration for better programming in Python

 

After the setup you will notice that the editor is now showing a point in place of space (figure 4b). And it has a guide line in the indentation of the document (figure 4b)

Figure 4. Text editor configured a) without the settings explained b) editor after the settings

 

Key binds

The atom has a series of keyboard shortcuts for the most diverse functions. You can see all of this in the settings document (Figure 1) in the keybinds tab. Some shortcuts that are already set up and may be useful for Python programmers are:

  • Select text + tab: causes selected text to indent
  • Select text + shift-tab: causes the selected text to return an indentation
  • select text + ctrl- /: causes the selected text to become comment. If it is a comment it makes it code
  • ctrl + s: save the document

Plugins to install

Now we will install the plugins that will further help your development in Python. To install, simply go to the configuration file (figure 1) and go to the install tab. In it enter the name of the plugins indicated in the following subtitles:

1-Plugin: atom-python-run to run python direct from Atom

The atom-python-run plugin is for running programs in python from the text editor.
To configure this plugin go to the settings document (figure 1), go to the packages tab find this plugin and click on the setting button of it. Clicking on it goes
you will be directed to the document of this plugin and in the field “F5 command” the path where the python you want to run is installed. Usually the
Python is installing in “C:\Users\[username]\AppData\Local\Programs\Python”.

2- Plugin: autocomplete-python to add atom python autocomplete.

This is the Python autocomplete plugin. For the installation of this plugin will appear the options between jedi and keti. Choose the jedi option:

Figure 5. Installation options for the autocomplete-python plugin a) jedi b) keti

 

After installing the auto-completion plugin you will notice that in a menu next to the writing cursor the python functions that can be auto-completed while you write your code (Figure 6). If this menu closes you can also use the shortcut “ctrl + space” to open it.

Figure 6. Auto complete menu of Atom code

 

3- Plugin: python-tools

The python-tools plugin is for running programs in python to move between functions. To do this just be in a document “* .py” with the python extension and click:

  • cursor over function / variable + ctrl-alt-g: make cursor is for function definition / variable
  • cursor over the definition of a function + ctrl-alt-u: highlights all uses of the chosen function
  • select text + ctrl + /: causes the selected text to become comment. If it is a comment it makes it code
  • ctrl + s: save the document
  • ctrl + space: open the upper snipete complete

4-Plugin: hyperclick

When installing the plugin hyperclick and the Plugin: python-tools is already installed, it generates a new shortcut. The new shortcut causes you to go to function definition using the ctrl-click shortcut in the desired function.

5-Plugin: minimap

The minimap plugin will enable on the right side of the document a mini-map of the entire open document.

6-Plugin: minimap-highlight-selected

The minimap-highlight-selected plugin enables the highlight text feature in the minimap of any text that the user selects. This helps a lot to find variables, functions in code.

7-Plugin: Magic Python

The language python plugin, which is installed, is already able to highlight language syntax Python programming languages in defined Python (.py) files. However, for some new language syntaxes this plugin can not properly highlight. In some tutorials here of the site we will use these new syntaxes. For that we highlight the Magic Python plugin. To make it work you should install this package as you did with the others. However, after installing you will have to go to the packages tab and disable the package language python that is installed at the factory.

Ready! You have a super text editor

Now you have the Atom IDE for Python that will help you optimize your work. Now it’s time to use this editor. If you are new to Python, go to Python introductory classes. If you already have more familiarity go to the opencv tutorials.
